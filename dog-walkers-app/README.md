# 🐶 Dog Walkers App

Live Demo: https://jprobussr.github.io/dog-walkers-react-app/

## Overview

The Dog Walkers App is a responsive React application that allows users to browse dog walkers, view their details, and interact with a clean, user-friendly interface. The focus of this project is on component-based architecture, UI structure, and real-world front-end development practices.

## Motivation

I built this project to simulate a real-world service-based application and strengthen my front-end development skills using React.

Instead of following a tutorial, I wanted to:

- Take an idea from concept to deployment
- Practice structuring a React application with reusable components
- Improve my ability to design clean, responsive user interfaces
- Gain experience working through real debugging and deployment challenges

This project reflects my goal of becoming a professional front-end developer by building practical, user-focused applications.

## Features

- Browse dog walkers
- Reusable React components
- Clean and responsive layout
- Organized project structure
- Deployed with GitHub Pages using GitHub Actions

## Tech Stack

- React (Vite)
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub
- GitHub Actions (deployment)

## What I Learned

- Building and structuring React components
- Managing project layout and UI composition
- Debugging real-world issues during deployment
- Configuring GitHub Actions for React apps
- Understanding how build tools like Vite work
